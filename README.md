# 🌐 Personal Portfolio Website

### Saif Atef — March 2026

## 📌 Overview

This project is a modern, responsive personal portfolio website designed to showcase my skills as a Front-End Developer.

As a 16-year-old high school student from Damanhour, El-Beheira, Egypt, I developed this website to present my background, technical expertise, featured projects, and contact information in a clean and professional way.

The website includes a sleek UI with smooth animations, dark/light mode support, mobile-friendly navigation, real-time project search, and a validated contact form. It represents my current capabilities in modern web development.

---

## 🎯 Objectives

* Build a visually appealing and fully responsive personal portfolio.
* Showcase projects with live demos and GitHub repositories.
* Implement modern UI/UX features such as theme switching and smooth scrolling.
* Demonstrate proficiency in HTML, CSS, and JavaScript.

---

## 🛠️ Technologies Used

### Core Technologies

* **HTML5** — Semantic structure and layout
* **CSS3** — Custom styling and advanced layouts
* **JavaScript** — Interactive functionality

### Frameworks & Libraries

* **Tailwind CSS (CDN)** — Utility-first styling for rapid development
* **AOS (Animate on Scroll)** — Scroll-based animations
* **Font Awesome** — Icons
* **Google Fonts (Poppins)** — Typography

### Tools & Practices

* **Git & GitHub** — Version control and hosting
* **Responsive Design** — Mobile-first approach

> ⚠️ Note: This project does not include a backend. The contact form uses client-side validation only.

---

## 🚀 Setup Instructions

### 1. Download the Project

Clone the repository or download it as a ZIP file.

### 2. Run the Website

Simply open:

```
index.html
```

### 3. Development

* Open the project folder in **VS Code** (recommended)
* Edit `index.html` (styles and scripts are included via CDN)
* Refresh your browser to view changes

> 💡 Make sure image paths are correctly linked based on your local folder structure.

---

## 🧩 Project Structure

### 🔹 Main Sections

* **Head** — Meta tags, Tailwind CDN, Font Awesome, AOS, and custom styles
* **Navbar** — Sticky navigation with theme toggle and responsive menu
* **Hero Section** — Introduction with background and call-to-action
* **About Section** — Personal info and bio
* **Tech Stack** — Skills grid (HTML, CSS, JS, Tailwind, Bootstrap, Python, Git, GitHub, WordPress)
* **Projects Section** — Featured projects with filtering search
* **Contact Section** — Form with validation and social links
* **Footer** — Copyright

---

### ⚙️ JavaScript Features

* Dark/Light mode toggle (with `localStorage`)
* Mobile menu toggle
* Real-time project search (by title & description)
* Back-to-top button
* Contact form validation (name, email, message)

---

### 🎨 Styling Approach

* Tailwind CSS utility classes for layout and responsiveness
* Custom CSS for hover effects and interactions
* Dark mode using Tailwind’s `dark:` prefix
